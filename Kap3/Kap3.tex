\chapter{Objetivos}

Para este capitulo los objetivos has sido obtenidos partiendo de la definición del problema que se presenta en el árbol de problemas\ref{fig:ArProblem}, en el cual se define el Problema general, luego sus principales causas y para finalizar las sub-causas. El árbol de problemas cumple la función de ser el punto de partida para la creación del árbol de objetivos \ref{fig:ArObjetivos}, que es finalmente el que proporciona tanto el objetivo general como los específicos para este capitulo y que posteriormente hará posible definir algunas de las actividades que estarán en el desarrollo de la metodología del proyecto.\\

\section{Objetivo General}

\begin{itemize}
\item Diseñar, desarrollar e implementar un software que permita un fácil acceso al procesamiento de datos satelitales del GNSS en formato RINEX.
\end{itemize}

\section{Objetivos específicos}
\begin{itemize}
\item Establecer las condiciones para una aplicación que no tenga costo alguno y de licencia libre, que permita procesamiento de datos GNSS en formato RINEX.\\
\item Selección de un algoritmo apropiado para procesar datos satelitales, explicación teórica del mismo y búsqueda de coherencia del algoritmo con el desarrollo del software.\\
\item Desarrollo de un software tipo web, para facilitar el acceso a los usuarios y que la capacidad de procesamiento no dependa de estos.\\
\end{itemize}
